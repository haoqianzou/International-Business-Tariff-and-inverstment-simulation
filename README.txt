Investment Portfolio Simulation - Usage Guide

This game is a simulation of an investment portfolio management system. Players can manage their investment portfolios across multiple rounds, making strategic asset allocation decisions.

Key Features:
  1. Multiple rounds of investment decision-making
  2. Real-time portfolio performance tracking
  3. Dynamic market conditions and events
  4. Interactive charts and visualizations
